# pokedex
Pokemon Index With Infinite Scroll and Detail Pokemon

### Data Source
Pokedex GraphiQL Console [here](https://graphql-pokemon.now.sh/)

### How To Run Local
```bash
# initial state
git clone https://github.com/fjfalah/pokedex.git
yarn
```
then
```bash
# development mode
yarn dev
```
or
```bash
# production mode
yarn build
yarn start
```

### Main Dependencies
- Typescript
- Next JS
- React JS
- Apollo GraphQL
- Jest (Test)
- Styled Components
